# Introduction

This is the Bootstrap Theme customized for the Hannover Re.

# Getting Started

To get started:

1. Install packages (`yarn install`)
2. Build Theme (`yarn build`)

To run the example: navigate to directory `bootstrap-samples`

1. Install packages (`yarn install`)
2. Serve the application (`ng serve`)
