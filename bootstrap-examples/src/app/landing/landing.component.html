Application style guide

Hannover Re uses a reduced and unpretentious design which gives the user rapid guidance and facilitates recognition. The aim is to achieve the greatest possible visual uniformity for all digital interfaces. Applying clear design principles also serves to improve usability.

This style guide concerns all screen-based media, such as

  • software products marketed externally to customers
  • home-made or licensed software products for in-house use
  • online tools and web-based services for in-house use

For the most recent application style guide please check here.

Fonts

The font ‘Compatil’ is used in all applications of Hannover Re. If ‘Compatil’ is not available or too costly (e.g. for only internally used applications) the free font ‘Source’ is used. To use the fonts, please get in touch with Corporate Communications. The Bootstrap Theme comes with the ‘Source‘ font included. Please use the fonts as described in the style guide.

Icons

To place functional icons (for standard functions such as Print, Search, Trashcan/Delete) Hannover Re holds a Font Awesome licence. Please get in touch with Corporate Communications to learn more. The media database Cumulus further provides a pool of visualisation icons for use in applications. Please use the icons as described in the style guide.
